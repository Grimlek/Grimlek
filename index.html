The Game of Pig

This project is one of the best examples of my expertise with object oriented programming in Java. The project was a school assignment in CS5044 at Virginia Tech and a very interesting one. The Game of Pig is a command line interface program in which the user plays against the computer. Both the user and the computer take turns rolling a pair of dice. The first player to a hundred points wins the game. For more game specific details please see the project outline.

The graphical user interface was provided as a third party library and we had to use object oriented programming to implement the artifical intelligence. Please see the project outline for more details.

Adventure Game

The project was based on the idea of creating a text based game similar to the first games ever made for computers, Zork and Infocom being the two most popular during that era. We were able to use alot of creativity for the project but still had a specific set of requirements we had to meet. I decided to do a text based dungeon crawler where you had to find the treasure room and collect the treasure. While trying to find the treasure room you had to fight your way through a numerous amount of enemies with your hero. For more project specific details please see the project outline.

TSCO Application

TSCO Application was a GUI application specifically implemented for reducing the number of excel files used for inventory control and providing a single location for access of all the data. The application provided additional features that multiple excel spreadhsheets couldn't such as search features, data validation and notifications. The intention of the project was to help TSCO achieve easier employee training and an application to provide easier access to data. The TSCO Application was never used in production due to the lack of interest in upper management of Tube Specialties but personally it was a great programming experience. For more project specific details please see the project outline.
